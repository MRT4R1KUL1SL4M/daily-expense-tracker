# 💰 Daily Expense Tracker

## 📌 Overview

A sleek and minimal expense tracking web app designed to manage daily finances efficiently.

## 🎯 Goal

To help users track spending habits and maintain financial awareness.

## 🛠️ Technologies Used

* HTML
* Tailwind CSS
* JavaScript

## ⚙️ Features

* Add & track expenses
* Local storage data persistence
* Clean and modern UI (Apple-style design)

## 🚀 How to Run

Open `index.html` in your browser.

## 🌱 Future Improvements

* Add charts & analytics
* Export data feature
* User authentication
